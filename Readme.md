# _Rush_ (Rust Shell)

## A simple shell with basic features in rust
Run it locally
```bash
git clone https://github.com/nrjais/rush.git && cd rush
cargo run
```

### What works
* Simple commands
* Variable expansion

### Builtins
* cd
* exit
* let
